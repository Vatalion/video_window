# Story 15.3: Policy Configuration
**Epic:** 15
**Points:** 5
**Status:** Ready
## User Story
As a policy operations manager, I want to configure enforcement rules from an admin interface so that the moderation engine reflects up-to-date compliance requirements.
## Acceptance Criteria
- Configure policy rules
- Automated detection
- Rule versioning
- Admin UI
## Tasks
- Model policy rules with version metadata and publish them through Serverpod endpoints consumed by enforcement services.
- Build the admin UI flow that allows safe creation, testing, and activation of policy changes with audit tracking.
- Wire automated detection hooks to pull the active rule set and evaluate violations in near real time.
