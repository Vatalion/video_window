# Story 15.1: Admin Dashboard
**Epic:** 15
**Points:** 8
**Status:** Ready
## User Story
As a trust and safety admin, I want a consolidated dashboard of platform health metrics so that I can triage issues and enforce policies quickly.
## Acceptance Criteria
- Dashboard with metrics
- Search/filter content
- Quick moderation actions
- Audit trail
## Tasks
- Build Serverpod endpoints that aggregate moderation, revenue, and engagement metrics needed for the dashboard widgets.
- Implement the Flutter admin dashboard with filtering, search, and action triggers gated behind admin RBAC guards.
- Persist quick-action events to an auditable log and surface the history within the dashboard for compliance review.
