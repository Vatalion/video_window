# Story 16.3: DSAR Workflows
**Epic:** 16
**Points:** 8
**Status:** Ready
## User Story
As a privacy operations manager, I want automated data subject access request workflows so that we can fulfill export and deletion obligations within regulatory timelines.
## Acceptance Criteria
- Data export (30-day GDPR)
- Deletion workflows
- Compliance tracking
- Audit trail
## Tasks
- Implement intake endpoints and queue processing that orchestrate export or deletion actions per DSAR type.
- Build operator tooling that shows DSAR status, SLA countdowns, and documentation for each request.
- Persist audit events and produce compliance dashboards demonstrating completion rates within the mandated window.
