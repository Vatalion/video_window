# Story 16.1: Secrets & RBAC
**Epic:** 16
**Points:** 5
**Status:** Ready
## User Story
As a platform engineer, I want centralized secrets management with fine-grained access controls so that sensitive credentials stay secure and auditable.
## Acceptance Criteria
- AWS Secrets Manager integration
- 90-day rotation
- RBAC audit logging
- Access control
## Tasks
- Provision AWS Secrets Manager resources and rotation jobs for environment credentials under infrastructure-as-code.
- Implement RBAC policies and Serverpod middleware that gate secret reads to approved service accounts and staff roles.
- Emit audit logs for every secret access and rotation event into the observability pipeline for compliance visibility.
