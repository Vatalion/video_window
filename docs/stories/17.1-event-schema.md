# Story 17.1: Event Schema
**Epic:** 17
**Points:** 5
**Status:** Ready
## User Story
As a data analyst, I want a consistent event schema with session context so that downstream analytics and growth experiments rely on trustworthy telemetry.
## Acceptance Criteria
- Segment SDK integration
- Event taxonomy
- User/session tracking
- Data validation
## Tasks
- Define the canonical analytics event dictionary and publish schema specs for product and engineering teams.
- Instrument the Flutter client and backend services with Segment SDK calls that populate user and session identifiers.
- Add validation jobs that flag malformed events and provide feedback channels for correcting instrumentation gaps.
