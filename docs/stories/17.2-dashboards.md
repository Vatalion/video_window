# Story 17.2: KPI Dashboards
**Epic:** 17
**Points:** 8
**Status:** Ready
## User Story
As a growth strategist, I want real-time KPI dashboards so that I can monitor funnel health and respond to performance shifts immediately.
## Acceptance Criteria
- BigQuery ETL
- Looker dashboards
- KPIs: DAU, GMV, conversion
- Real-time updates
## Tasks
- Build ETL jobs that land cleaned event data in BigQuery with hourly refreshes for key KPIs.
- Model Looker explores and dashboards reflecting DAU, GMV, conversion, and offer activity with drill-down filters.
- Implement alerting or highlight thresholds that surface significant KPI changes to the strategy team.
