# Story 15.2: Listing Takedown
**Epic:** 15
**Points:** 8
**Status:** Ready
## User Story
As a trust and safety reviewer, I want to remove violating listings and notify sellers so that harmful content leaves the marketplace immediately.
## Acceptance Criteria
- Remove listings instantly
- Strike system
- Seller notification
- Evidence archival
## Tasks
- Create moderation commands that disable listings, record strike counts, and snapshot evidence artifacts.
- Extend admin tooling to trigger takedowns with reason codes and automatically notify the seller with appeal instructions.
- Update persistence layer to archive evidence and track strike history for repeat offender policies.
